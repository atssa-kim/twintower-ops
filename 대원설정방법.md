# 트윈타워 상황대응 앱 — 대원 설정 방법

> 재난 발령 시 즉시 알림을 받으려면 아래 순서대로 설정하세요.

---

## 1단계. 앱 열기

스마트폰 브라우저(Chrome 권장)에서 아래 주소를 입력합니다.

```
https://atssa-kim.github.io/twintower-ops/
```

---

## 2단계. 홈 화면에 추가 (바탕화면 바로가기 설치)

앱을 홈 화면에 설치해야 **백그라운드 알림**이 정상 동작합니다.

### Android (Chrome)
1. 브라우저 우측 상단 **⋮ 메뉴** 탭
2. **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
3. 설치 완료 후 홈 화면 아이콘으로 실행

### iPhone / iPad (Safari)
> ⚠️ iOS는 반드시 **Safari**로 열어야 합니다. Chrome 앱으로는 설치 불가.

1. Safari에서 위 주소 접속
2. 하단 **공유 버튼** (□↑) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **추가** 탭
5. 홈 화면의 아이콘으로 실행 **(이후 항상 아이콘으로 열 것)**

> ⚠️ iPhone은 iOS 16.4 이상이어야 알림이 지원됩니다.

---

## 3단계. 로그인

1. 앱 실행
2. **사번 선택** — 본인 사번을 드롭다운에서 선택
3. **로그인** 버튼 탭

---

## 4단계. 발령 알림 켜기

로그인 후 화면 상단에 아래 배너가 표시됩니다.

```
🔔 발령 알림 받기 — 눌러서 켜기
```

1. 노란 배너를 **탭**
2. "알림 허용" 팝업 → **허용** 선택
3. "✅ 알림이 켜졌습니다" 메시지 확인

> 이 단계를 완료해야 재난 발령 시 푸시 알림이 옵니다.

---

## 5단계. 설정 완료 확인

| 항목 | 확인 방법 |
|------|-----------|
| 앱 설치 | 홈 화면에 🔔 아이콘 존재 |
| 로그인 | 상단에 내 이름·소속 표시 |
| 알림 설정 | 노란 배너가 사라짐 |

---

## 발령 시 사용 방법

1. 푸시 알림 수신 → 알림 탭 → 앱 자동 실행
2. 또는 앱 아이콘 직접 실행
3. 화면에 **재난 발령 내용** 및 **내 임무 목록** 자동 표시
4. 임무 수행 시 항목을 **탭하여 체크** ✓
5. 지휘자 화면에서 임무 수행률 실시간 확인 가능

---

## 자주 묻는 질문

**Q. 알림 배너가 안 보여요.**
- Firebase 연결(LIVE 모드)일 때만 표시됩니다. 상단 LIVE 배지를 확인하세요.

**Q. iPhone에서 알림이 안 와요.**
- Safari로 접속 후 홈 화면에 추가했는지 확인하세요.
- iOS 16.4 미만은 웹 푸시를 지원하지 않습니다.

**Q. 알림이 차단됐다고 나와요.**
- 스마트폰 설정 → 알림 → Safari(또는 Chrome) → 해당 사이트 알림 **허용**으로 변경

**Q. 사번이 목록에 없어요.**
- 관리자에게 사번 등록을 요청하세요.
